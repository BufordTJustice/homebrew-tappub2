# BufordTJustice Tappub2

## How do I install these formulae?

`brew install bufordtjustice/tappub2/<formula>`

Or `brew tap bufordtjustice/tappub2` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "bufordtjustice/tappub2"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
